# Countdown Timer

A simple and tiny timer that counts down in big figlet numbers. \
Made for personal use, do whatever you want with this.

## Usage

``` sh
$ ./countdown 100
  _    ___     ___
 / |  / _ \   / _ \
 | | | | | | | | | |
 | | | |_| | | |_| |
 |_|  \___/   \___/
```



## Build

``` sh
git clone --recurse-submodules https://github.com/LainLayer/countdown-timer
./build.sh
```
